
*** HTML 5 ***

___________________________________________________________________________________________________________________________________________

* <!DOCTYPE>:
- It is an instruction to the web browser about what version of HTML the page is written in.
- HTML 5 - <!DOCTYPE html>.
- HTML 4.01 Strict.
- HTML 4.01 Transitional.
- HTML 4.01 Frameset.

___________________________________________________________________________________________________________________________________________

* <meta>:
- The <meta> tag provides metadata about the HTML document. Metadata is data (information) about data.

___________________________________________________________________________________________________________________________________________


** New features in HTML 5:

* Semantic elements:
- <article>
- <aside>
- <footer>
- <header>
- <main>
- <nav>
- <section>

* New Input Types:
- color
- date
- email
- number
- range

* New Input Attributes:
- autocomplete
- autofocus
- height and width
- min and max
- pattern
- placeholder
- required
- step

* New Media Elements:
- <audio>
- <video>
- <embed>

* New Graphic Elements:
- <svg>
- <canvas>

* New HTML 5 APIs:
- HTML Geolocation
- HTML Local Storage
- HTML Web Workers
